## Qr Code Bot
### A  telegram bot to generate qr code
## Dependencies
- pyqrcode\
  ```pip3 install pyqrcode```
 - pypng\
 ```pip3 install pypng```
- python telegram wrapper\
  ```pip3 install python-telegram-bot```

## Available commands
- ```/start``` - Starts the bot.

- ```/help``` - For help message.
